Size of Duplication Chain (SDC)

This metric measures the total size of a duplication chain in terms of lines of code. A duplication chain is composed of a number of smaller islands of exact clones interrupted by small fragments of non-identical code. The size of the duplication chain is the most relevant metric, because it measures the area of code that contains code duplication.
