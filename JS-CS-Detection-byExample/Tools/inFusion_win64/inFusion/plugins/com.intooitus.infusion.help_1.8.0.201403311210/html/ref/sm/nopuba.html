Number of Public Attributes (NOPUBA)

This metric is defined as the number of public attributes of a class. A public attribute cancels the level of protection from the users of a class, for that attribute. If a public attribute is used, more logic has to be added to test for invalid values up front. This means more effort when designing a class, as well as the fact that it is more computationally expensive. Public attributes also "clutter" the interface. The fewer public attributes you present, the "cleaner" and easier your object will be to work with. Private attributes assure flexibility while providing an ease of use that would otherwise not be there.
