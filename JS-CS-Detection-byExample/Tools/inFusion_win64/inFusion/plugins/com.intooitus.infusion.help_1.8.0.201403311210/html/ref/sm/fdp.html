Foreign Data Providers (FDP)

This metric is defined as the distinct number of other capsules (classes in case of attributes, modules in case of global variables) from which data (attributes and/or global variables) are accessed by the measured operation. This metric is concerned with the coupling between capsules. This represents a major aspect of the object oriented design and also of procedural design, since the reuse degree, the maintenance and testing effort for a capsule are decisively influenced by the coupling level between capsules.
