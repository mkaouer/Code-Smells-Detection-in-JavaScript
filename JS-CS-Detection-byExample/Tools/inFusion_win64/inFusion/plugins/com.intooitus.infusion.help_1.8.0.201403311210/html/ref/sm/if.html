Instability Factor (IF)

This metric is defined as the ratio of efferent coupling (Ce) to total coupling, which is the sum of efferent coupling (Ce) and afferent coupling (Ca). This metric is an indicator of the capsule's (class or module) resilience to change. Ce represents the number of capsules used (called or accessed) by the measured subsystem that are located in other subsystems. Ca represents the number of capsules that are located outside the measured subsystem and use (call or access) capsules located in the measured subsystem.
