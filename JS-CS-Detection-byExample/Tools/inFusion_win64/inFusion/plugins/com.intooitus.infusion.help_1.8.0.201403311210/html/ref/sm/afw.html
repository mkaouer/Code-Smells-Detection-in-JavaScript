Average Function Weight (AFW)

This metric is defiend as the average static complexity of all functions in a module. The metric CYCLO (McCabe's cyclomatic number) is used to quantify the functions's complexity. Large AFW values indicate the presence of few but very complex functions. Low numbers can point to data modules, that only hold data and provide no functionality.
