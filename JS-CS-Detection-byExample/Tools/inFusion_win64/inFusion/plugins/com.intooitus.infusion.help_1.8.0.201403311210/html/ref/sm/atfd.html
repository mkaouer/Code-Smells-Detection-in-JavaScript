Access to Foreign Data (ATFD)

This metric is defined as the number of distinct data (attributes and/or global variables) accessed by an operation (method or function), whereby the data are external to the capsule of the operation (class in the case of methods, module in the case of functions). Accessing the structured global, shared data space of another capsule is a violation of the locality principle of good software design.
