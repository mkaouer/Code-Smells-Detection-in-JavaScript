Number of Added Services (NAS)

This metric is defined as the number of new public methods added (defined) by a derived class. The normal expectation for a subclass is to extend the functionality inherited from the superclass and this includes adding new services. The NAS should usually decrease as you move through the layers of the hierarchy. The lower threshold is one, since all classes should have at least one method to justify their existence. The upper threshold is a declining scale, based on the class hierarchy nesting level.
