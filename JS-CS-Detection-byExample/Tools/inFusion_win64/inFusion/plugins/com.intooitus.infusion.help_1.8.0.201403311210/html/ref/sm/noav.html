Number of Accessed Variables (NOAV)

This metric is defined as the number of distinct variables accessed in the body of an operation. This metric counts the number of data variables (e.g. local variables, parameters, class attributes, etc) that a method's body accesses. It is a measure of the connectedness and therefore complexity of the method's implementation.
