Number of Methods (NOM)

This metric is defined as the total number of methods defined in a class. This includes both instance methods and class methods. This metric gives us information about the functional size of the class.

  Known thresholds
Absolute threshold: 20 instance methods for model classes + 4 class methods, 40 instance methods for UI classes + 4 class methods.
Relative threshold: Number of class methods / Number of instance methods = 0.2.
