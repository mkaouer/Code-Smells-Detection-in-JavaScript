Map Spotlight

The Map Spotlight is an intuitive simple to use tool for interacting with Polymetric Maps. It consists of a search bar, where the user can write simple highlight conditions, and a table area where the entities matching the condition are shown. These entities are also marked in the map using a dashed green border. In its default state the Map Spotlight is hidden and can be activated by clicking on its corresponding button in the toolbar of the Visual Exploration Area, shown in the figure below.



Highlight conditions can be specified using the simple syntax described below:

highlight_condition = name_matching_condition
                    | design_flaw_filtering_condition
                    | metric_filtering_condition

name_matching_condition = [entity_type] <regular_expression>

design_flaw_filtering_condition = [entity_type] is <design_flaw_name> | <filter_name>

metric_filtering_condition = [entity_type] has metric comp_operator <numerical_value>

metric = <design_flaw_name>
       | <metric_name>

comp_operator = < | <= | = | >= | >

In the above conditions <entity_type> means any of the entity types appearing in a Polymetric Map (package, class, module, method, function, attribute, or variable), while <metric_name> represents any of the supported metrics applicable to the entity type in question.
