Lines of Code (LOC)

This metric is defined as the number of lines of code of an operation, including blank lines and comments. The LOC metric is useful in the case of a number of different software activities, such as estimating, planning, tracking the progress of projects, normalizing measures of productivity and defect densities, and deriving measures and effectiveness of reuse strategies.
