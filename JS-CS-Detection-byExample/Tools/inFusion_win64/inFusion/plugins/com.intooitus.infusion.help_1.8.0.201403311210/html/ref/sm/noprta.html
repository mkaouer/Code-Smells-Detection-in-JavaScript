Number of Protected Attributes (NOPRTA)

This metric is defined as the number of protected attributes of a class. Protected data affects code maintainability, as a protected attribute reveals a piece of the internal structure of a class to its subclasses. Free access to a class's data is dangerous in terms of the class's invariants.
