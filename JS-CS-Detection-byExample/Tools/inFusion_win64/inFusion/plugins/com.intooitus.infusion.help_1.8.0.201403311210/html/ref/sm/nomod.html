Number of Modules (NOMOD)

This metric is defined as the number of files that define global functions. A module is a file that defines at least one function. Modules are the procedural programming equivalent of classes. In C++, a file can contain one or more classes, and if the file contains at least one function then that file also contains one module (made up of all the other functions and global variables that are not defined in the class / classes). A file can contain one module at most.
