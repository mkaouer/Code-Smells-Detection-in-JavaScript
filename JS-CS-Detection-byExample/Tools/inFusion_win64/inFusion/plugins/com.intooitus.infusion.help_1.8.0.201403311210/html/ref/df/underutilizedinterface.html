Underutilized Interface

Underutilized Interface is a design flaw inspired by Martin Flowler's code smell Speculative Generality, which refers in essence to the mistake of over-engineering the design of a system, so as to prepare it for handling special requirements that never seem to materialize. Should those situations occur the design would be appropriate, but the unnecessary capabilities clutter the design with additional complexity, making it harder to maintain.

Detection rule

An underutilized (module) interface is one that defines a significant amount of unused public operations. The detection rule for Underutilized Interface looks for large interfaces, for which less than half of their public operations (functions with external linkage) are actually called from outside the containing module.
