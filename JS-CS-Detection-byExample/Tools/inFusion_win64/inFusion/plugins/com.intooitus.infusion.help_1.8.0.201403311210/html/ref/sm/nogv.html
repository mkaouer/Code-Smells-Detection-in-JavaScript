Number of Global Variables (NOGV)

This metric is defined as the number of all top-level externally visible global variables defined in a module (i.e. all global variables, whose visibility scope isn't limited to the current source file). Overexposing too many data can cause maintenance problems, in the sense that otherwise local changes to the layout of a module might break dependent modules and result in changes in many other places. This metric is rather often mistaken for the metric Number of Variables (NOV), which counts all global variables, regardless of visibility scope.
