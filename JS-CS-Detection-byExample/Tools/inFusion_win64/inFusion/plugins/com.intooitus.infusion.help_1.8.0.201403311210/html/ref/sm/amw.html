Average Method Weight (AMW)

This metric is defiend as the average static complexity of all methods in a class. The metric CYCLO (McCabe's cyclomatic number) is used to quantify the method's complexity. Large AMW values indicate the presence of few but very complex methods and is typically a sign of bad object-oriented style. Low numbers can point to data classes, that only hold data and provide no functionality.
