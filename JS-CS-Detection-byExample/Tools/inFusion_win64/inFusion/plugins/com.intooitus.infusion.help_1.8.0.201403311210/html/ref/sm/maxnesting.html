Maximum Nesting Level (MAXNESTING)

This metric is defined as the maximum nesting level of control structures within an operation. This a measure of the depth of nesting of control structures (e.g. if-statements, loops) in a program. Deeply nested control structures are hard to understand and are potentially error-prone.
