Software Metrics

Software metrics are used for quantifying specific aspects related to software development. They can be divided into product metrics, quantifying aspects of the software product itself, and process metrics, quantifying aspects of the development process. While the literature includes a huge number of metrics, not all of them are equally useful when it comes to assessing design and code quality in general.

We use a catalog of carefully selected product metrics for quantifying both procedural and object-oriented code and design. They are the building blocks of the detection rules for design flaws.
