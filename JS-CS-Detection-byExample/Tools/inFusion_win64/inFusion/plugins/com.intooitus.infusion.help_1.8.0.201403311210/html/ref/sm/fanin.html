Fan-In (FANIN)

This metric counts all distinct operations that call the measured operation. If an operation X is called twice by the operation Y then X has a FANIN value of 1. A high value for FANIN means that operation X is tightly coupled to the rest of the design and changes to X will have extensive ripple effects.
