Specialization Index (SPIDX)

This metric is defined as the number of overridden methods multiplied by the depth of the overriding method in the class hierarchy and divided by the metric NOM (number of methods) of the initial class. The SPIDX metric quantifies the quality of sub-classing, by measuring the extent of functionality of a class that is replaced in its subclasses. Ideally, a subclass should only specialize a superclass, meaning that it should only add new functionality and not replace existing one. Excessive overriding usually indicates that the abstraction represented by the superclass is inappropriate.
