Base-class Usage Ratio (BUR)

This metric is defined as the number of inherited members (i.e. members that are defined as protected in the base class) used by the measured class, divided by the total number of inherited members from the base class. A low ratio indicates that the subclasses are using fewer members than the base class is providing access to. This can be either the case when these members can be encapsulated, without breaking the sub classes, or the base class should be split in two ore more sub classes.
