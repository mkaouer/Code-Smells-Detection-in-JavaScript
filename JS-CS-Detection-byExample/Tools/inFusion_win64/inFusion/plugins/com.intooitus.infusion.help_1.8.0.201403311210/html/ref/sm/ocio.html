Outgoing Coupling Intensity for an Operation (OCIO)

This metric is defined as the number of distinct operations (methods or functions) from other (external) classes or modules called by the measured operation. This metric is similar to FANOUT, the difference being that it does not count operations that belong to the same container. This metric is useful in assessing how much an operation depends on others.

OCIO_POLY is a version of OCIO that treats in a special manner the case of a polymorphic call (i.e. "virtual" C++ methods that are overridden in one or more classes). For this case OCIO_POLY will count both the method in the base class, where the called methods was defined, as well as all methods that redefine the method in a polymorphic manner.
