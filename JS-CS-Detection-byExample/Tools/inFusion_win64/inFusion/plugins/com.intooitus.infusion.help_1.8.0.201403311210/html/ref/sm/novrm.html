Number of Overriding Methods (NOVRM)

This metric is defined as the number of methods in a class that are overriding (or specializing) inherited methods. A too low value indicates either superfluous inheritance or too wide member scopes. A high value indicates lack of inheritance or heavy use of overriding and specialization.
