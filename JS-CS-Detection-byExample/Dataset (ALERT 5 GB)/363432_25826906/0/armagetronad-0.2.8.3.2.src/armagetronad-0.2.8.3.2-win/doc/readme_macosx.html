Mac OS X Usage

Installation

Simply double click the Armagetron dmg file to mount it. Then, drag the Armagetron icon to your chosen install location.

Play

 Double click the 'Armagetron' icon.

Uninstall

 Drag the 'Armagetron' icon to the trash.

Configuration

 On OS X, Armagetron's configuration files (user.cfg, etc) can be found in "/Library/Application Support/Armagetron"

Dedicated Server

 The dedicated server has not yet been built for OS X, wait for the next release..
