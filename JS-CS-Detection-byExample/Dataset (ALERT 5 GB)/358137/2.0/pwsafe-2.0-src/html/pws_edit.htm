Edit/View Entry

In this window, you can change the information you've stored for an entry. By default, the password will be replaced with a line of asterisks. To view your existing password, hit "Show Password." (This can be changed via the "Show password by default in edit mode" option.

For a description of the fields in this dialog box, see the Add Entry help topic.

If you make a mistake and want to leave this window without saving your changes, hit "Cancel." To save changes, click the "OK" button.
