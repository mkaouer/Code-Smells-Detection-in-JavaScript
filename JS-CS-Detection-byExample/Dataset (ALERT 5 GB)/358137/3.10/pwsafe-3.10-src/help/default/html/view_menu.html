View Menu

The View Menu controls the available display formats. You can switch between Classic and New Toolbar Icons as well as how their user names and passwords are organized: either as a single list (Flattened List Mode) or organized by category (Nested Tree Mode). You can also expand or collapse the groups displayed in Nested Tree Mode, as well as select the font in which the entries are displayed.

Switching Between Menu Icon Views

  1. Go the the View menu
  2. Select Classic Toolbar or New Toolbar

Switching Between Nested Tree and Flattened List Modes

  1. Go the the View menu
  2. Select Nested Tree Mode or Flattened List Mode

Expand All

Expands all groups, making all entries visible (Nested Tree Mode only).

Collapse All

Collapses all groups (Nested Tree Mode only).

Change Font

Select a different font for displaying entries.
