Menu Toolbar

There are two different views available for toolbar icons: new and classic. Each of the icons visible on the toolbar provide pop up tips. Simply hover your mouse pointer over the icon and a small tip will appear. For convenience, this topic provides a quick reference for each of the icons.

Switching Between Menu Icon Views

  1. Go the the View menu
  2. Select Classic Toolbar or New Toolbar
