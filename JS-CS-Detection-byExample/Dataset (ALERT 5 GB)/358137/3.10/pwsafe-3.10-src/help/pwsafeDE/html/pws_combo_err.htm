Falsches Datenbank-Paßwort

Das Fenster mit dem Titel "Incorrect Passkey" erscheint, wenn Sie das Paßwort für die Datenbank dreimal hintereinander falsch eingegeben haben. Falls Sie sich bei der Eingabe nur vertippt haben, können Sie es noch einmal versuchen (Try Again). Sie können auch eine andere Datenbank öffnen (Open Another Database...) oder eine komplett neue Datenbank erstellen (Create a New Database).

Falls Sie das Paßwort für die Datenbank vergessen haben sollten, stellen Sie das Öffnen der Datenbank erst einmal zurück. Vielleicht fällt Ihnen das richtige Paßwort später irgendwann wieder ein. Leider kann Ihnen nicht geholfen werden, wenn Sie das Paßwort vergessen haben. Die Sicherheit dieses Programms beruht nämlich auf der Tatsache, daß die Datenbank ohne Kenntnis des richtigen Paßwortes nicht geöffnet werden kann.
