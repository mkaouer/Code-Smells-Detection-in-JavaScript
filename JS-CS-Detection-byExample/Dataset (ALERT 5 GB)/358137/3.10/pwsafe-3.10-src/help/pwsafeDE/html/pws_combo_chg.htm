Datenbank-Paßwort ändern

Im Fenster "Key Change" können Sie das Paßwort für die momentan geöffnete Datenbank ändern. Zunächst müssen Sie das alte Paßwort einmal eingeben, danach das neue Paßwort zweimal. Die Eingabe wird mit Sternchen verdeckt. Das Paßwort ist sensibel für Groß- und Kleinschreibung, d.h. "hallo" ist nicht das gleiche wie "HALLO". Zum Abschluß klicken Sie auf "OK". Die "OK"-Schaltfläche wird nur dann aktiviert sein, wenn Sie das alte Paßwort sowie das neue Paßwort (zweimal) korrekt eingegeben haben.
