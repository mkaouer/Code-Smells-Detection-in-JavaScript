Allen Einträgen einen Benutzernamen zuweisen?

Dieser Dialog erscheint wenn Sie eine Passwortdatenbank geöffnet haben, bei der für die einzelnen Einträge kein Benutzername angegeben wurde.

Klicken Sie auf "Yes", um einen Benutzernamen einzugeben, der allen Einträgen zugeordnet werden soll. Klicken Sie auf "No", um die Datenbank so zu lassen, wie sie ist.
