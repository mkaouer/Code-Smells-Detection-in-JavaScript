Das Programmfenster

Das Programmfenster von Password Safe besteht aus der Menüleiste, der darunter liegenden Werkzeugleiste und einer Darstellung der gerade geöffneten Datenbank.



1. Die Menüleiste

Die Menüleiste hat folgende Einträge (Tastaturkürzel werden, soweit vorhanden, [in Klammern] hinter dem Eintrag angezeigt.):


1.1 Menü "File"

Hier können Sie Datenbanken öffnen, speichern und exportieren.

New Database [STRG+N] - Eine neue Paßwort-Datenbank erstellen.

Open Database [STRG+O] - Öffnet eine anderen Paßwort-Datenbank. Falls notwendig, wird das Programm Sie fragen, ob die gerade geöffnete Datenbank gespeichert werden soll. Welche Datenbank gerade geöffnet ist, kann in der Titelleiste des Hauptfensters abgelesen werden.

Save [STRG+S] - Die gerade geöffnete Datenbank speichern.

Save As - Die gerade geöffnete Datenbank kann unter einem anderen Namen abgespeichert werden. Die Datenbank unter dem alten Namen bleibt weiterhin bestehen.

Export To - Exportiert die gerade geöffnete Datenbank in ein altes Format von Password Safe (Old (1.x) format). Sie können auch in eine Textdatei exportieren (Plain text), die mit Hilfe eines Texteditors oder einer Tabellenkalkulation geöffnet werden kann.

Exit - Beendet Password Safe.


1.2 Menü "Edit"

Zum Bearbeiten, Löschen oder Suchen von Datenbankeinträgen. Darüber hinaus können Sie Paßwörter und Benutzernamen in die Zwischenablage kopieren.

Add Entry [STRG+A] - Fügt einen Eintrag der gerade geöffneten Datenbank hinzu.

Edit/View Entry [STRG+Enter] - Öffnet den gerade markierten Eintrag, um ihn detailliert anzuzeigen und bearbeiten zu können.

Delete Entry [ENTF] - Löscht den gerade markierten Eintrag.

Rename Entry [F2] - Umbenennen des gerade markierten Eintrages (nur in der Baumansicht)

Find Entry [STRG+F] - Öffnet eine Suchmaske, um alle Datenbankeinträge (ausgenommen Paßwörter) nach einem Stichwort zu durchsuchen. Die Anzahl der gefundenen Ergebnisse (matches) wird angezeigt. Ist das Stichwort öfter als einmal vorhanden, können durch klicken auf "Find Next" nacheinander die entsprechenden Einträge markiert werden. Standardmäßig wird zwischen Groß- und Kleinschreibung unterschieden. Dies kann durch Aktivieren von "Case sensitive search" geändert werden.

Add Group - Erstellt eine neue Gruppe mit den Namen "New Group" (nur in der Baumansicht). Die Gruppe wird automatisch ausgewählt, so eine andere Gruppenbezeichnung direkt eingegeben werden kann.

Copy Password to Clipboard [STRG+C] - Um das Paßwort eines Eintrages in die Zwischenablage zu kopieren, doppelklicken Sie im Hauptfenster auf den entsprechenden Eintrag. Sie können den Eintrag auch mit einem Klick der linken Maustaste markieren und in der Menüleiste "Edit" -> "Copy Passwort to Clipboard" auswählen. Oder klicken Sie in der Symbolleiste auf das entsprechende Symbol.

Nun können Sie das Paßwort in das Paßwortfeld eines Programms oder einer Webseite einfügen. Drücken Sie dazu die rechte Maustaste über einem Paßwortfeld und wählen Sie dann den "Einfügen"-Befehl. (Falls mit der rechten Maustaste kein Menü zum Einfügen anzeigt wird, können Sie statt dessen auch mit der linken Maustaste in das Paßwortfeld klicken und anschließend die Tastenkombination CTRL-V bzw. STRG-V oder SHIFT-INF bzw. SHIFT-EINF drücken.)

Standardmäßig wird das Paßwort aus der Zwischenablage gelöscht, wenn Sie Passwort Safe minimieren oder beenden.

Copy Username to Clipboard [STRG+U] - Identisch mit "Copy Password to Clipboard", nur daß nicht das Paßwort, sondern der Benutzernamen in die Zwischenablage kopiert wird.

Clear Clipboard [STRG+ENTF] - Löscht das Paßwort, das sich gerade in der Zwischenablage befindet. Hat ein anderes Programm Daten in die Zwischenablage kopiert, werden diese nicht gelöscht.

Perform Auto type [STRG+T] - Mit der Autotype-Funktion können Zeichenfolgen an das zuletzt aktive Fenster gesendet werden. So können Sie den Benutzernamen und das Paßwort direkt in z.B. das Anmelde-Formular einer Webseite eingeben lassen, ohne die Kopieren- und Enfügen-Befehle verwenden zu müssen. Das Aufrufen des Autotype-Befehls bewirkt:

  1. Das Programmfenster von Password Safe wird minimiert.
  2. Der Benutzername des gerade markierten Eintrages wird in das aktive Eingabefeld des zuletzt aktiven Fensters eingetragen.
  3. Das Drücken der Tabulator-Taste wird simuliert, um in das nächste Eingabefeld zu gelangen.
  4. Das Paßwort wird eingefügt.
  5. Das Drücken der Enter-Taste wird simuliert.
Die Autotype-Funktion kann konfiguriert werden, indem Sie im "Notes"-Feld des entsprechenden Eintrages folgenden Text einfügen:

    autotype:\u\t\p\t\n

Mit diesem Kommando wird der Benutzername (\u) gesendet, das Drücken der Tabulator-Taste simuliert (\t) und anschließend das Paßwort gesendet (\p). Nach einem weiteren Tabulator (\t) wird das Drücken der Enter-Taste simuliert (\n). Diese Einstellungen werden standardmäßig ausgeführt. Wollen Sie beispielsweise nur das Paßwort übergeben, müßten Sie folgenden Text verwenden: "autotype:\p\t\n". Mit "\r" können Sie das Drücken der Return-Taste simulieren.

Zusätzlich kann eine Verzögerung zwischen den einzelnen Zeichn mit "\dnnn" festgelegt werden. nnn ist die Verzögerung in Millisekunden. Der Standwert ist 10. Um eine halbe Sekunde zu warten und dann mit 10 Millisekunden pro Zeichen fortzusetzen, müßten Sie "\d500\d10" eingeben.

Hinweis: Um die Autotype-Funktion erfolgreich auszuführen, müssen Sie in der Regel vorher im Zielfenster den Cursor in das gewünschte Eingabefeld bringen.


1.3 Menü "View"

Anzeigen der Paßwörter als Liste (List) oder in der Baumansicht (Tree).


1.4 Menü "Manage"

Hier können Sie das Paßwort einer Datenbank ändern, Sicherheitskopien erstellen und die Programmeinstellungen ändern.

Change Safe Combination - Ändert das Paßwort der gerade geöffneten Datenbank. Geben Sie bei "Old Safe Combination" das alte Paßwort ein und bei "New Safe Combination" und "Confirm" das neue Paßwort. Zum Abschluß klicken Sie auf "OK". Die "OK"-Schaltfläche wird nur dann aktiviert sein, wenn Sie das alte Paßwort sowie das neue Paßwort (zweimal) korrekt eingegeben haben.

Make Backup - Erstellt eine Sicherheitskopie der gerade geöffneten Datenbank.

Restore from Backup - Stellt eine Sicherheitskopie wieder her. Soll die wiederhergestellte Datenbank weiter verwendet werden, so muß sie mit "File" -> "Save As" unter einem neuen Namen gespeichert werden.

Options - Genaue Informationen zu den Programmeinstellungen finden Sie hier.


1.5 Menü "Help"

Get Help - Zeigt diese Hilfedatei an.

Visit Password Safe website... - Lädt im Browser die Homepage von Password Safe.

About Password Safe... - Zeigt die Programmversion von Password Safe an.



2. Die Werkzeugleiste

Die Werkzeugliste besteht aus den kleinen Grafiken unter der Menüleiste, mit den die am häufigsten benötigten Befehle direkt ausgeführt werden können. Bewegen Sie den Mauszeiger über eine der Grafiken und Sie erhalten eine kurze Beschreibung der Funktion (in Englisch).



3. Darstellung der gerade geöffneten Datenbank

Abhängig davon, welche Ansicht gewählt wurde, erscheinen die Einträge der Datenbank als Liste oder in einer Baumansicht. In der Listenansicht können die Einträge durch ein Anklicken der Spaltenüberschriften nach Titel (Title), Benutzername (User Name) oder Paßwort (Password) alphabetisch sortiert angezeigt werden. In der Baumansicht werden die einzelnen Gruppen automatisch alphabetisch sortiert angezeigt.
