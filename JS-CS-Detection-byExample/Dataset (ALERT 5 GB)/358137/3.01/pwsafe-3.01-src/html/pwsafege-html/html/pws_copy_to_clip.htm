Passwort in die Zwischenablage kopieren

Dieser Dialog weist darauf hin, dass das gewählte Passwort in die Zwischenablage kopiert wurde. Sie können es nun direkt dort einfügen, wo Sie es benötigen, wie z.B. das Passwortfeld eines anderen Programms oder das Formularfeld einer Internetseite.

Wenn Sie das Passwort woanders einfügen möchten, kann es sein, dass der "Einfügen"-Befehl nicht verfügbar ist, wenn Sie die rechte Maustaste über einem Passwortfeld drücken. Sie können das Passwort aber dennoch einfügen, und zwar indem Sie mit der linken Maustaste in das Passwortfeld klicken und anschließend die Tastenkombination CTRL-V bzw. STRG-V oder SHIFT-INF bzw. SHIFT-EINF drücken.
