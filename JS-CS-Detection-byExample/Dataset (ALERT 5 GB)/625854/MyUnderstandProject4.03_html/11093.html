30     /**
31      * Defines a BNF-style grammar based on JSGF grammar rules in a file.
32      * <p/>
33      * <p/>
34      * The Java Speech Grammar Format (JSGF) is a BNF-style, platform-independent, and vendor-independent textual
35      * representation of grammars for use in speech recognition. It is used by the <a href="http://java.sun.com/products/java-media/speech/">Java
36      * Speech API (JSAPI) </a>.
37      * <p/>
38      * <p/>
39      * Here we only intend to give a couple of examples of grammars written in JSGF, so that you can quickly learn to write
40      * your own grammars. For more examples and a complete specification of JSGF, go to <p><a
41      * href="http://java.sun.com/products/java-media/speech/forDevelopers/JSGF/">http://java.sun.com/products/java-media/speech/forDevelopers/JSGF/
42      * </a>.
43      * <p/>
44      * <p/>
45      * <b>Example 1: "Hello World" in JSGF </b>
46      * <p/>
47      * <p/>
48      * The example below shows how a JSGF grammar that generates the sentences "Hello World":
49      * <p/>
50      * <p/>
51      * <table width="100%" cellpadding="10"> <tr> <td bgcolor="#DDDDDD">
52      * <p/>
53      * <pre>
54      *  #JSGF V1.0
55      * <p/>
56      *  public &lt;helloWorld&gt; = Hello World;
57      * </pre>
58      * <p/>
59      * </td> </tr> </table>
60      * <p/>
61      * <i>Figure 1: Hello grammar that generates the sentences "Hello World". </i>
62      * <p/>
63      * <p/>
64      * The above grammar is saved in a file called "hello.gram". It defines a public grammar rule called "helloWorld". In
65      * order for this grammar rule to be publicly accessible, we must be declared it "public". Non-public grammar rules are
66      * not visible outside of the grammar file.
67      * <p/>
68      * <p/>
69      * The location of the grammar file(s) is(are) defined by the {@link #PROP_BASE_GRAMMAR_URL baseGrammarURL}property.
70      * Since all JSGF grammar files end with ".gram", it will automatically search all such files at the given URL for the
71      * grammar. The name of the grammar to search for is specified by {@link #PROP_GRAMMAR_NAME grammarName}. In this
72      * example, the grammar name is "helloWorld".
73      * <p/>
74      * <p/>
75      * <b>Example 2: Command Grammar in JSGF </b>
76      * <p/>
77      * <p/>
78      * This examples shows a grammar that generates basic control commands like "move a menu thanks please", "close file",
79      * "oh mighty computer please kindly delete menu thanks". It is the same as one of the command & control examples in the
80      * <a href="http://java.sun.com/products/java-media/speech/forDevelopers/JSGF/">JSGF specification </a>. It is
81      * considerably more complex than the previous example. It defines the public grammar called "basicCmd".
82      * <p/>
83      * <p/>
84      * <table width="100%" cellpadding="10"> <tr> <td bgcolor="#DDDDDD">
85      * <p/>
86      * <pre>
87      *  #JSGF V1.0
88      * <p/>
89      *  public &lt;basicCmd&gt; = &lt;startPolite&gt; &lt;command&gt; &lt;endPolite&gt;;
90      * <p/>
91      *  &lt;command&gt; = &lt;action&gt; &lt;object&gt;;
92      *  &lt;action&gt; = /10/ open |/2/ close |/1/ delete |/1/ move;
93      *  &lt;object&gt; = [the | a] (window | file | menu);
94      * <p/>
95      *  &lt;startPolite&gt; = (please | kindly | could you | oh mighty computer) *;
96      *  &lt;endPolite&gt; = [ please | thanks | thank you ];
97      * </pre>
98      * <p/>
99      * </td> </tr> </table>
100      * <p/>
101      * <i>Figure 2: Command grammar that generates simple control commands. </i>
102      * <p/>
103      * <p/>
104      * The features of JSGF that are shown in this example includes: <ul> <li>using other grammar rules within a grammar
105      * rule. <li>the OR "|" operator. <li>the grouping "(...)" operator. <li>the optional grouping "[...]" operator. <li>the
106      * zero-or-many "*" (called Kleene star) operator. <li>a probability (e.g., "open" is more likely than the others).
107      * </ul>
108      * <p/>
109      * <p/>
110      * <h3>From JSGF to Grammar Graph</h3>
111      * <p/>
112      * After the JSGF grammar is read in, it is converted to a graph of words representing the grammar. Lets call this the
113      * grammar graph. It is from this grammar graph that the eventual search structure used for speech recognition is built.
114      * Below, we show the grammar graphs created from the above JSGF grammars. The nodes <code>"&lt;sil&gt;"</code> means
115      * "silence".
116      * <p/>
117      * <p/>
118      * <img src="doc-files/helloWorld.jpg"> <br> <i>Figure 3: Grammar graph created from the Hello World grammar. </i>
119      * <p/>
120      * <p/>
121      * <img src="doc-files/commandGrammar.jpg"> <br> <i>Figure 4: Grammar graph created from the Command grammar. </i>
122      * <p/>
123      * <p/>
124      * <h3>Limitations</h3>
125      * <p/>
126      * There is a known limitation with the current JSGF support. Grammars that contain non-speech loops currently cause the
127      * recognizer to hang.
128      * <p/>
129      * For example, in the following grammar
130      * <p/>
131      * <pre>
132      *  #JSGF V1.0
133      *  grammar jsgf.nastygram;
134      *  public <nasty> = I saw a ((cat* | dog* | mouse*)+)+;
135      * </pre>
136      * <p/>
137      * the production: ((cat* | dog* | mouse*)+)+ can result in a continuous loop, since (cat* | dog* | mouse*) can
138      * represent no speech (i.e. zero cats, dogs and mice), this is equivalent to ()+. To avoid this problem, the grammar
139      * writer should ensure that there are no rules that could possibly match no speech within a plus operator or kleene
140      * star operator.
141      * <p/>
142      * <h3>Dynamic grammar behavior</h3> It is possible to modify the grammar of a running application. Some rules and
143      * notes: <ul> <li> Unlike a JSAPI recognizer, the JSGF Grammar only maintains one Rule Grammar. This restriction may be
144      * relaxed in the future. <li> The grammar should not be modified while a recognition is in process <li> The call to
145      * JSGFGrammar.loadJSGF will load in a completely new grammar, tossing any old grammars or changes. No call to
146      * commitChanges is necessary (although such a call would be harmless in this situation). <li> RuleGrammars can be
147      * modified  via calls to RuleGrammar.setEnabled and RuleGrammar.setRule). In order for these changes to take place,
148      * JSGFGrammar.commitChanges must be called after all grammar changes have been made. </ul>
149      * <p/>
150      * <h3>Implementation Notes</h3> <ol> <li>All internal probabilities are maintained in LogMath log base. </ol>
151      */

258         /**
259          * Parses the given Rule into a network of GrammarNodes.
260          *
261          * @param rule the Rule to parse
262          * @return a grammar graph
263          */
264         private GrammarGraph parseRule(Rule rule) throws GrammarException {
265             GrammarGraph result;
266     
267             if (rule != null) {
268                 debugPrintln("parseRule: " + rule.toString());
269             }
270     
271             if (rule instanceof RuleAlternatives) {
272                 result = parseRuleAlternatives((RuleAlternatives) rule);
273             } else if (rule instanceof RuleCount) {
274                 result = parseRuleCount((RuleCount) rule);
275             } else if (rule instanceof RuleName) {
276                 result = parseRuleName((RuleName) rule);
277             } else if (rule instanceof RuleSequence) {
278                 result = parseRuleSequence((RuleSequence) rule);
279             } else if (rule instanceof RuleTag) {
280                 result = parseRuleTag((RuleTag) rule);
281             } else if (rule instanceof RuleToken) {
282                 result = parseRuleToken((RuleToken) rule);
283             } else if (rule instanceof RuleParse) {
284                 throw new IllegalArgumentException(
285                         "Unsupported Rule type: RuleParse: " + rule.toString());
286             } else {
287                 throw new IllegalArgumentException("Unsupported Rule type: "
288                         + rule.toString());
289             }
290             return result;
291         }

294         /**
295          * Parses the given RuleName into a network of GrammarNodes.
296          *
297          * @param initialRuleName the RuleName rule to parse
298          * @return a grammar graph
299          */
300         private GrammarGraph parseRuleName(RuleName initialRuleName)
301                 throws GrammarException {
302             debugPrintln("parseRuleName: " + initialRuleName.toString());
303             GrammarGraph result = (GrammarGraph) ruleStack.contains(initialRuleName
304                     .getRuleName());
305     
306             if (result != null) { // its a recursive call
307                 return result;
308             } else {
309                 result = new GrammarGraph();
310                 ruleStack.push(initialRuleName.getRuleName(), result);
311             }
312             RuleName ruleName = ruleGrammar.resolve(initialRuleName);
313     
314             if (ruleName == RuleName.NULL) {
315                 result.getStartNode().add(result.getEndNode(), 0.0f);
316             } else if (ruleName == RuleName.VOID) {
317                 // no connection for void
318             } else {
319                 if (ruleName == null) {
320                     throw new GrammarException("Can't resolve " + initialRuleName
321                             + " g " + initialRuleName.getFullGrammarName());
322                 }
323                 RuleGrammar rg = recognizer.getRuleGrammar(ruleName
324                         .getFullGrammarName());
325                 if (rg == null) {
326                     throw new GrammarException("Can't resolve grammar name "
327                             + ruleName.getFullGrammarName());
328                 }
329     
330                 Rule rule = rg.getRule(ruleName.getSimpleRuleName());
331                 if (rule == null) {
332                     throw new GrammarException("Can't resolve rule: "
333                             + ruleName.getRuleName());
334                 }
335                 GrammarGraph ruleResult = parseRule(rule);
336                 if (result != ruleResult) {
337                     result.getStartNode().add(ruleResult.getStartNode(), 0.0f);
338                     ruleResult.getEndNode().add(result.getEndNode(), 0.0f);
339                 }
340             }
341             ruleStack.pop();
342             return result;
343         }

346         /**
347          * Parses the given RuleCount into a network of GrammarNodes.
348          *
349          * @param ruleCount the RuleCount object to parse
350          * @return a grammar graph
351          */
352         private GrammarGraph parseRuleCount(RuleCount ruleCount)
353                 throws GrammarException {
354             debugPrintln("parseRuleCount: " + ruleCount);
355             GrammarGraph result = new GrammarGraph();
356             int count = ruleCount.getCount();
357             GrammarGraph newNodes = parseRule(ruleCount.getRule());
358     
359             result.getStartNode().add(newNodes.getStartNode(), 0.0f);
360             newNodes.getEndNode().add(result.getEndNode(), 0.0f);
361     
362             // if this is optional, add a bypass arc
363     
364             if (count == RuleCount.ZERO_OR_MORE || count == RuleCount.OPTIONAL) {
365                 result.getStartNode().add(result.getEndNode(), 0.0f);
366             }
367     
368             // if this can possibly occur more than once, add a loopback
369     
370             if (count == RuleCount.ONCE_OR_MORE || count == RuleCount.ZERO_OR_MORE) {
371                 newNodes.getEndNode().add(newNodes.getStartNode(), 0.0f);
372             }
373             return result;
374         }

377         /**
378          * Parses the given RuleAlternatives into a network of GrammarNodes.
379          *
380          * @param ruleAlternatives the RuleAlternatives to parse
381          * @return a grammar graph
382          */
383         private GrammarGraph parseRuleAlternatives(RuleAlternatives ruleAlternatives)
384                 throws GrammarException {
385             debugPrintln("parseRuleAlternatives: " + ruleAlternatives.toString());
386             GrammarGraph result = new GrammarGraph();
387     
388             Rule[] rules = ruleAlternatives.getRules();
389             float[] weights = ruleAlternatives.getWeights();
390             normalizeWeights(weights);
391     
392             // expand each alternative, and connect them in parallel
393             for (int i = 0; i < rules.length; i++) {
394                 Rule rule = rules[i];
395                 float weight = 0.0f;
396                 if (weights != null) {
397                     weight = weights[i];
398                 }
399                 debugPrintln("Alternative: " + rule.toString());
400                 GrammarGraph newNodes = parseRule(rule);
401                 result.getStartNode().add(newNodes.getStartNode(), weight);
402                 newNodes.getEndNode().add(result.getEndNode(), 0.0f);
403             }
404     
405             return result;
406         }

409         /**
410          * Normalize the weights. The weights should always be zero or greater. We need to convert the weights to a log
411          * probability.
412          *
413          * @param weights the weights to normalize
414          */
415         private void normalizeWeights(float[] weights) {
416             if (weights != null) {
417                 double sum = 0.0;
418                 for (int i = 0; i < weights.length; i++) {
419                     if (weights[i] < 0) {
420                         throw new IllegalArgumentException("negative weight");
421                     }
422                     sum += weights[i];
423                 }
424                 for (int i = 0; i < weights.length; i++) {
425                     if (sum == 0.0f) {
426                         weights[i] = LogMath.getLogZero();
427                     } else {
428                         weights[i] = logMath.linearToLog(weights[i] / sum);
429                     }
430                 }
431             }
432         }

435         /**
436          * Parses the given RuleSequence into a network of GrammarNodes.
437          *
438          * @param ruleSequence the RuleSequence to parse
439          * @return the first and last GrammarNodes of the network
440          */
441         private GrammarGraph parseRuleSequence(RuleSequence ruleSequence)
442                 throws GrammarException {
443     
444             GrammarNode startNode = null;
445             GrammarNode endNode = null;
446             debugPrintln("parseRuleSequence: " + ruleSequence);
447     
448             Rule[] rules = ruleSequence.getRules();
449     
450             GrammarNode lastGrammarNode = null;
451     
452             // expand and connect each rule in the sequence serially
453             for (int i = 0; i < rules.length; i++) {
454                 Rule rule = rules[i];
455                 GrammarGraph newNodes = parseRule(rule);
456     
457                 // first node
458                 if (i == 0) {
459                     startNode = newNodes.getStartNode();
460                 }
461     
462                 // last node
463                 if (i == (rules.length - 1)) {
464                     endNode = newNodes.getEndNode();
465                 }
466     
467                 if (i > 0) {
468                     lastGrammarNode.add(newNodes.getStartNode(), 0.0f);
469                 }
470                 lastGrammarNode = newNodes.getEndNode();
471             }
472     
473             return new GrammarGraph(startNode, endNode);
474         }

525         /** Commit changes to all loaded grammars and all changes of grammar since the last commitChange */
526         public void commitChanges() throws IOException {
527             try {
528                 if (loadGrammar) {
529                     recognizer = new BaseRecognizer();
530                     recognizer.allocate();
531                     ruleGrammar = recognizer.loadJSGF(baseURL, grammarName);
532                     ruleGrammar.setEnabled(true);
533                     loadGrammar = false;
534                 }
535     
536                 recognizer.commitChanges();
537                 ruleStack = new RuleStack();
538                 newGrammar();
539     
540                 firstNode = createGrammarNode("<sil>");
541                 GrammarNode finalNode = createGrammarNode("<sil>");
542                 finalNode.setFinalNode(true);
543     
544                 // go through each rule and create a network of GrammarNodes
545                 // for each of them
546     
547                 String[] ruleNames = ruleGrammar.listRuleNames();
548                 for (int i = 0; i < ruleNames.length; i++) {
549                     String ruleName = ruleNames[i];
550                     if (ruleGrammar.isRulePublic(ruleName)) {
551                         String fullName = getFullRuleName(ruleName);
552                         GrammarGraph publicRuleGraph = new GrammarGraph();
553                         ruleStack.push(fullName, publicRuleGraph);
554                         Rule rule = ruleGrammar.getRule(ruleName);
555                         GrammarGraph graph = parseRule(rule);
556                         ruleStack.pop();
557     
558                         firstNode.add(publicRuleGraph.getStartNode(), 0.0f);
559                         publicRuleGraph.getEndNode().add(finalNode, 0.0f);
560                         publicRuleGraph.getStartNode().add(
561                                 graph.getStartNode(), 0.0f);
562                         graph.getEndNode().add(publicRuleGraph.getEndNode(), 0.0f);
563                     }
564                 }
565                 postProcessGrammar();
566             } catch (EngineException ee) {
567                 // ee.printStackTrace();
568                 throw new IOException(ee.toString());
569             } catch (GrammarException ge) {
570                 // ge.printStackTrace();
571                 dumpGrammarException(ge);
572                 throw new IOException("GrammarException: " + ge);
573             } catch (MalformedURLException mue) {
574                 throw new IOException("bad base grammar url " + baseURL + " "
575                         + mue);
576             }
577         }

626         /** Represents a graph of grammar nodes. A grammar graph has a single starting node and a single ending node */
627         class GrammarGraph {
628     
629             private GrammarNode startNode;
630             private GrammarNode endNode;
631     
632     
633             /**
634              * Creates a grammar graph with the given nodes
635              *
636              * @param startNode the staring node of the graph
637              * @param endNode   the ending node of the graph
638              */
639             GrammarGraph(GrammarNode startNode, GrammarNode endNode) {
640                 this.startNode = startNode;
641                 this.endNode = endNode;
642             }
643     
644     
645             /** Creates a graph with non-word nodes for the start and ending nodes */
646             GrammarGraph() {
647                 startNode = createGrammarNode(false);
648                 endNode = createGrammarNode(false);
649             }
650     
651     
652             /**
653              * Gets the starting node
654              *
655              * @return the starting node for the graph
656              */
657             GrammarNode getStartNode() {
658                 return startNode;
659             }
660     
661     
662             /**
663              * Gets the ending node
664              *
665              * @return the ending node for the graph

667             GrammarNode getEndNode() {

673         /** Manages a stack of grammar graphs that can be accessed by grammar name */
674         class RuleStack {
675     
676             private List stack;
677             private HashMap map;
678     
679     
680             /** Creates a name stack */
681             public RuleStack() {
682                 clear();
683             }
684     
685     
686             /** Pushes the grammar graph on the stack */
687             public void push(String name, GrammarGraph g) {
688                 stack.add(0, name);
689                 map.put(name, g);
690             }
691     
692     
693             /** remove the top graph on the stack */
694             public void pop() {
695                 map.remove(stack.remove(0));
696             }
697     
698     
699             /**
700              * Checks to see if the stack contains a graph with the given name
701              *
702              * @param name the graph name
703              * @return the grammar graph associated with the name if found, otherwise null
704              */
705             public GrammarGraph contains(String name) {
706                 if (stack.contains(name)) {
707                     return (GrammarGraph) (GrammarGraph) map.get(name);
708                 } else {
709                     return null;
710                 }
711             }
712     
713     
714             /** Clears this name stack */
715             public void clear() {
716                 stack = new LinkedList();
717                 map = new HashMap();
718             }
719         }
