La fenêtre principale de Password Safe

La fenêtre principale contient la barre de menus, la barre d'outils, et les enregistrements de la base ouverte.

La barre de Menus

La barre de menu contient les menus suivants (lorsqu'un raccourci clavier existe, il est mentionné à côté du menu [comme ceci] :

File (Fichiers)

Pour lire ou écrire une base de données PasswordSafe. Vous pouvez créer une nouvelle base de données, en ouvrir une, sauver la base ouverte, avec son nom actuel ou un autre nom, exporter la basse de données, soit avec l'ancien format de la version 1.7 ou sous forme de fichier texte lisible par un éditeur de texte ou un tableur.

New Database (Nouvelle Base de données) [Ctrl+N]

Créer une nouvelle base de données. Vous devrez entrer une combinaison pour cette base, après quoi une nouvelle base sans nom sera créée.

Open Database (Ouvrir Base de données) [Ctrl+O]

Sélectionner ce menu pour ouvrir une autre base de données. Le programme vous demandera de sauvegarder la base actuellement ouverte si nécessaire/si vous le désirez. Vous pouvez visualiser le nom de la base actuellement ouverte en haut de la fenêtre principale.

Save (Sauver) [Ctrl+S]

Sélectionner ce menu pour sauver la Base de données de mots de passe.

Save As (Sauvegarder sous)

Sélectionner ce menu pour sauvegarder la base en cours sous un autre nom. La version sous l'ancien nom existera toujours (sans les changements les plus récents).

Edit (Editer)

Pour ajouter ou supprimer un enregistrement, changer un enregistrement, trouver un enregistrement contenant un texte déterminé, ou manipuler le Presse papier, tel que décrit ci-dessous :

Add Entry (Ajouter un enregistrement) [Ctrl+A]

Sélectionner ce menu pour ajouter un mot de passe dans la base en cours.

Edit/View Entry (Editer/Voir un enregistrement) [Ctrl+Enter]

Sélectionner ce menu pour modifier ou visualiser l'enregistrement sélectionné.

Delete Entry (Supprimer un enregistrement) [Del]

Pour supprimer un enregistrement, sélectionner son titre et choisir ce menu.

Rename Entry (Renommer un enregistrement) [F2]

(Dans la vue Arbre seulement) Cette commande vous permet de directement modifier le nom de l'enregistrement sélectionné.

Find Entry (Rechercher un enregistrement) [Ctrl+F]

Cette commande vous permet de taper un texte à rechercher dans toute votre base de mots de passe, pour tous les champs texte (excepté les mots de passe eux-mêmes). Le nombre d'enregistrement correspondant est alors affiché. Si plus d'un enregistrement est trouvé, le bouton Find Next sélectionnera chaque enregistrement correspondant. Par défaut, la recherche est sensible à la casse. Ceci peut être modifié en cochant la case Case sensitive search.

Add Group (Ajouter un groupe)

(Dans la vue arbre seulement) Cette commande ajoutera un nouveau groupe, nommé par défaut New Group. Le nouveau groupe est automatiquement sélectionné, et le nom est directement modifiable.

Copy Password to Clipboard (Copier le mot de passe vers le Presse papier) [Ctrl+C]

Pour copier un mot de passe vers le Presse papier, faire un double-clic sur le titre de l'enregistrement. Vous pouvez également sélectionner le titre puis choisir le menu Copy Password to Clipboard. Vous pouvez ensuite coller le mot de passe dans n'importe qu'elle fenêtre. (Si l'option Coller n'est pas disponible, vous pouvez toujours coller en utilisant le raccourci clavier Control-V ou Shift-Inser.
Par défaut, le Presse papier sera effacé lorsque vous quittez ou minimisez Password Safe.

Copy Username to Clipboard (Copier l'utilisateur vers le Presse papier) [Ctrl+U]

Pour copier un utilisateur vers le Presse Papier, sélectionner le titre et sélectionner le menu Copy Username to Clipboard. Vous pouvez alors coller l'utilisateur dans n'importe qu'elle fenêtre. (Si l'option Coller n'est pas disponible, vous pouvez toujours coller en utilisant le raccourci clavier Control-V ou Shift-Inser.
Par défaut, le Presse papier sera effacé lorsque vous quittez ou minimisez Password Safe.

Clear Clipboard (Effacer le Presse Papier) [Ctrl+Del]

Pour effacer sûrement un mot de passe du Presse papier, sélectionner le menu Clear Clipboard. Si une autre application que Password Safe a copiée des données dans le Presse papier, il ne sera pas effacé.

Perform Auto type (Exécute l'auto remplissage) [Ctrl+T]

L'option Autotype vous permet d'envoyer directement des frappes de touches vers la fenêtre derrière Password Safe. Ceci vous permet d'entrer votre nom d'utilisateur et mot de passe dans des formulaires Internet, des applications, etc. sans utiliser le copier coller. La commande Autotype (depuis le menu Edit ou en utilisant Ctrl+T) exécute les tâches suivantes :
  1. Minimise PasswordSafe
  2. Colle le nom de l'utilisateur actuellement sélectionné vers la fenêtre actuelle (la dernière fenêtre active utilisée avant Password Safe)
  3. Envoie une Tabulation vers le champ suivant
  4. Colle le mot de passe
  5. Simule la frappe de la touche Entrée.
Notez que ce comportement peut être personnalisé en ajoutant du texte spécialement formaté dans le champ Notes d'un enregistrement comme suit :
    autotype:\u\t\p\t\n

Ceci enverra l'Utilisateur (\u) puis une Tabulation (\t) puis le mot de Passe (\p) puis une deuxième tabulation et enfin la touche Entrée (\n) - ceci est le comportement par défaut. Pour envoyer, par exemple, uniquement le mot de Passe, le texte suivant doit être ajouté "autotype:\p\t\n". '\r' enverra la touche Return.

Vous devez positionner le curseur dans la fenêtre de texte dont vous avez besoin avant d'activer cette fonctionnalité.

View (Voir)

Pour sélectionner le type d'affichage des enregistrements. Soit sous forme de liste soit d'arbre hiérarchique. Utiliser également ce menu pour sélectionner entre les deux types de barres d'outils (Nouvelle ou Classique).

Manage (Gérer)

Pour changer la combinaison de votre base de données (mot de passe Principal), faire une copie de sauvegarde ou restaurer depuis une copie de sauvegarde, et pour paramétrer certaines options.

Change Safe Combination (Changer la combinaison)

Sélectionner ce menu pour changer la combinaison de Password Safe pour la base de données ouverte. Saisissez la combinaison dans la fenêtre intitulée Safe Combination puis à nouveau (pour confirmation) dans la fenêtre marquée Verify. Enfin cliquer sur le bouton OK qui ne sera actif que si la saisie a été identique les deux fois.

Make Backup (Faire une sauvegarde)

Choisir ce menu pour faire une sauvegarde de la base de données de mot de passe ouverte.

Restore from Backup (Restaurer depuis une sauvegarde)

Sélectionner ce menu pour restaurer une base de données depuis une sauvegarde. La version restaurée le sera sans nom. Vous devez lui donner un nom dans l'option Save As si vous souhaitez la conserver.

Options

Help (Aide)

Help - Pour accéder à l'aide en ligne, la page du projet sur Internet et afficher le numéro de version de PasswordSafe.

The Toolbar (La barre d'Outils)

La barre d'outils contient des boutons de raccourcis vers les fonctions les plus utiles. Déplacer la souris au dessus d'un bouton pour obtenir une courte description de la fonction.

The Entries (Les enregistrements)

En fonction du type de vue sélectionnée, les enregistrements sont affichés sous forme de liste simple ou sous forme d'arbre. Lorsqu'ils sont affichés en liste, les enregistrements peuvent être triés en cliquant sur le nom du champ qui servira de critère de tri. Dans l'affichage Arbre, les enregistrements peuvent être rangés par catégorie, sous catégorie, etc., permettant ainsi de regrouper ensemble des mots de passe.
