Program Credits

Grateful thanks to Bruce Schneier for creating the Twofish algorithm and for releasing Password Safe to the public; and to SourceForge.net for hosting the project and supporting the Open Source community.
Password Safe version 3.06 and later uses TinyXML for parsing the application's configuration file. The code for TinyXML is provided under the zlib license.

The links that follow lead to various pages on the SourceForge web site related to the Password Safe project:

  • Development Team: Just some of the folks that have helped develop and keep Password Safe going.
  • Password Safe Forum: A forum for questions, support, and problems.
  • Password Safe News: RSS feeds available for this project.
  • Password Safe's low volume (roughly one message every month or so) spam free announcement and user mailing list page.
