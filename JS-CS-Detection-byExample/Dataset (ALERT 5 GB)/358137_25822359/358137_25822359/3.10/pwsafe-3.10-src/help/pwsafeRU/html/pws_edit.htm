Edit/View Entry

В этом окне вы можете изменить информацию, которую сохранили для какой-либо записи. По умолчанию пароль будет заменен линией звездочек. Чтобы увидеть ваш пароль, нажмите "Show Password." (Это может быть изменено в опции "Show password by default in edit mode".)

Описания полей в диалоговом окне даны в разделе справки Add Entry .

Если вы хотите оставить это окно без изменений, введенных вами по ошибке, нажмите "Cancel." Чтобы сохранить изменения, нажмите кнопку "OK" .
