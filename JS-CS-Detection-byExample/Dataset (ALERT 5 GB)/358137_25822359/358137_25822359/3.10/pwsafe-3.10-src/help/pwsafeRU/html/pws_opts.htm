Опции Password Safe

Настраиваемые опции разделены на следующие категории:
Display - опции, относящиеся к тому, как отображается PasswordSafe
Security - опции, имеющее отношение к безопасности программы
Password Policy - Определяют параметры для автоматически генерируемых паролей
Username - Определяет имя пользователя по умолчанию для новых записей
Misc - Разные настройки

Display

Always keep PasswordSafe on top - При выборе этой опции PasswordSafe всегда будет виден.
Show Password in display list - Если выбрать эту опцию, то при отображении записей в главном окне в виде списка будет добавлена дополнительная колонка, показывающая пароль для каждой записи. Включать эту опцию не рекомендуется, так как открывается потенциальная брешь в системе безопасности.
Show password by default in edit mode - Выбор этой опции приведет к тому, что пароли будут показываться в диалоговом окне Edit/View Entry. По умолчанию показывается ряд звездочек ('*****'), пока вы не нажмете кнопку "Show Password".

Security

Clear the clipboard upon minimize or exit - Если вы выберете эту опцию, буфер обмена будет очищен при сворачивании или выходе из PasswordSafe.

Lock password database on minimize - Если вы выберете эту опцию, секретная информация (такая, как ваша сейфовая комбинация и названия записей) будет стерта из памяти, когда Password Safe свернут. Когда вы восстановите Password Safe, он запросит у вас сейфовую комбинацию.

Confirm password database save on minimize - Если вы выберете эту опцию, Password Safe спросит вас о необходимости сохранения изменений в базе данных при сворачивании. Если эта опция не выбрана, база данных будет сохранена автоматически.

Confirm item copy to clipboard - Если выбрать эту опцию, Password Safe известит вас, когда пароль будет скопирован в буфер обмена. После того, как вы привыкнете к тому, как работает Password Safe, вам, скорее всего, захочется отключить эту опцию.

Password Policy

Default password length - Это поле определяет длину пароля, генерируемого при нажатии кнопки "Generate Random Password" в диалоговых окнах Edit/View Entry и Add Entry. Минимальная длина -- 4 символа, максимальная -- 1024.

Use lowercase letters - Выберите эту опцию, если хотите, чтобы сгенерированный пароль содержал буквы нижнего регистра.

Use UPPERCASE letters - Выберите эту опцию, если хотите, чтобы сгенерированный пароль содержал буквы верхнего регистра.

Use digits - Выберите эту опцию, если хотите, чтобы сгенерированный пароль содержал цифры (0-9).

Use symbols (i.e., ,%,$, etc.) - Выберите эту опцию, если хотите, чтобы сгенерированный пароль содержал и другие символы, помимо букв и цифр.

Use only easy-to-read characters (i.e., without "0" and "O") - Выбор этой опции приведет к тому, что сгенерированный пароль не будет содержать визуально похожих символов, таких, как цифра ноль и заглавная буква О. Это особенно полезно, если вы не копируете и вставляете пароли, а предпочитаете посмотреть их и вводить вручную.

Username

Use a default username - Если выбрана эта опция, имя пользователя, введенное в поле ниже, будет автоматически вставляться в поле Username диалогового окна Add Entry. Также, если поле Username оставлено незаполненным, будет вставлено имя пользователя по умолчанию.

Query user to set username as default - Если выбрана эта опция, каждый раз, когда вы вводите имя пользователя для новой записи, а имя пользователя по умолчанию не задано, вам будет предлагаться сделать его именем пользователя по умолчанию. Смотрите выше дополнительную информацию об именах пользователя по умолчанию.

Misc

Confirm deletion of items - Когда выбрана эта опция, PasswordSafe будет запрашивать подтверждение на удаление выбранной записи, когда вы выберете команду Delete Entry.

Save database immediately after Edit or Add - Выбор этой опции заставит PasswordSafe сохранять изменения в базе данных немедленно, а не ждать команд Save или Save As.
