Hilfedatei installieren

Um die deutsche Hilfedatei "pwsafege.chm" vollständig zu installieren, kopieren Sie sie in das Programmverzeichnis von Password Safe. Benennen Sie die englische Hilfedatei "pwsafe.chm" nach "pwsafe.eng" um. Anschließend benennen Sie die deutsche Hilfedatei "pwsafege.chm" nach "pwsafe.chm" um.
