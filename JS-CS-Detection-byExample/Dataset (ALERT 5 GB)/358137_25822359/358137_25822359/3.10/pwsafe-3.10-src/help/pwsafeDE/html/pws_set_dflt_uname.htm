Als Standard-Benutzernamen festlegen?

Hier können Sie den gerade eingegebenen Benutzernamen als Standard-Benutzername (default username) festlegen. Der Standard-Benutzername wird nicht im Hauptfenster angezeigt und automatisch in das Feld für den Benutzernamen eingetragen, wenn Sie der Datenbank einen neuen Eintrag hinzufügen.

Die Einstellungen zum Standard-Benutzernamen können Sie in den Programmeinstellungen ändern.
