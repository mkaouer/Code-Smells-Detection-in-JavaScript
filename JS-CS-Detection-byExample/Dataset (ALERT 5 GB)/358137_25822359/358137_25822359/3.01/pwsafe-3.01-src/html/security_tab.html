Security Tab

The Security Tab provides access to user controlled security settings. Ease of use and program security are affected by the options listed here, but the underlying encrypted password database is not.

Clear clipboard upon minimize or exit

When checked, the program clears clipboard contents of any stored data (such as a password) when the interface is minimized or closed. This prevents anyone else or any other program from reading the contents of the clipboard.

Lock password database on minimized

When checked, locks the password database when the program is minimized. When maximized again for use, a password is required to access the program.

Confirm password database save on minimize

Checking this choice causes Password Safe to request user confirmation of any unsaved changes when the program is minimized. Password Safe will continue to request user confirmation of unsaved changes when the program is exited.

Confirm item copy to clipboard

When checked, a prompt is displayed requesting user confirmation every time a password is copied from the database to the clipboard.

Lock password database on workstation lock

Checking this causes the password database to lock when the workstation is locked, preventing unauthorized access by users that may have access to the workstation.

Lock password database after....

Checking this option and setting a time causes Password Safe to lock the password database due to non-use after a set interval.
