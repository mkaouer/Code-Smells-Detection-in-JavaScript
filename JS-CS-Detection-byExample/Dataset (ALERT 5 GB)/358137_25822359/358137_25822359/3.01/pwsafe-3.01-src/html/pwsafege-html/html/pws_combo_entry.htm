Datenbank-Paßwort eingeben

Das Fenster "Safe Combination Entry" sehen Sie immer dann, wenn Sie Password Safe starten, wenn Sie eine neue Datenbank öffnen oder wenn Sie eine Datenbank von einer Sicherheitskopie wiederherstellen. Geben Sie das Paßwort für die Datenbank ein und klicken Sie auf "OK". Das Paßwort ist sensibel für Groß- und Kleinschreibung, d.h. "hallo" ist nicht das gleiche wie "HALLO".

Falls das Paßwort dreimal falsch eingegeben wurde, erscheint das "Incorrect Passkey"-Fenster. Sie können hiermit eine andere Datenbank öffnen oder eine neue erstellen.
