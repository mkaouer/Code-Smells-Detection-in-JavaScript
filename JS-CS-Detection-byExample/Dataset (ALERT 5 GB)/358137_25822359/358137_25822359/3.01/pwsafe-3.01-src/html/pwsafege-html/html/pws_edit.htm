Eintrag bearbeiten/anzeigen

In diesem Fenster (Titel "Edit/View Entry") können Sie die zu einem Eintrag gehörenden Angaben ändern.

Standardmäßig wird das Paßwort nicht im Klartext anzeigt, sondern durch Sternchen ('*****') verdeckt. Um das Paßwort im Klartext anzuzeigen, klicken Sie auf "Show Password". (Sie können auch alternativ "Manage" -> "Options" -> "Show password by default in edit mode" aktivieren.)

Haben Sie bei der Eingabe einen Fehler gemacht und wollen Sie das Fenster ohne zu speichern verlassen, klicken Sie auf "Cancel". Um die Änderungen zu speichern, klicken Sie auf "OK".

Für eine Beschreibung der einzelnen Eingabefelder schauen Sie bitte unter dem Hilfethema "Datenbank-Eintrag hinzufügen" nach.
