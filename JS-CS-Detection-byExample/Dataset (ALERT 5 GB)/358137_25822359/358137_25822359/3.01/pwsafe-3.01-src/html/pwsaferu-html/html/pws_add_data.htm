Add Entry

В этом окне вы можете добавить новую запись в текущую БД. В этом окне есть следующие поля:

 Поле Group позволяет вам организовать ваши записи в логические группы. Все записи с одинаковым текстом в этом поле будут показаны в одной группе при выборе View -> Tree в главном окне программы. Заметьте, что можно определять иерархию групп с помощью отделения подгрупп точками. Например, "Финансы.Банки" поместит запись в подгруппу Банки группы Финансы. Это поле необязательное, и, если оставить его пустым, запись будет помещена в группу самого верхнего уровня.
Поле Title -- это имя, под которым запись отображается в главном окне. Это поле не может быть пустым.
Поле Username используется для хранения имени пользователя, относящегося к этой записи. Если вы назначите default username в диалоговом окне Options, оно появится здесь. В дальнейшем вы сможете скопировать имя пользователя в буфер обмена.
Поле Password -- это поле для ввода пароля, относящегося к данной записи. Если вы нажмете кнопку "Generate Random Password" , это поле будет заполнено паролем, случайно сгенерированным в соответствии с правилами, заданными в разделе Password Policy диалогового окна Options. Это поле не может быть пустым.
Вы можете использовать поле Notes для хранения любой другой информации, которую вам надо помнить об этой записи, -- например, адреса URL или телефонного номера. Если вы введете URL, начинающийся с "http://" или "https://", тогда вы сможете перейти к этому URL, кликнув правой кнопкой мыши по записи в главном окне программы.
