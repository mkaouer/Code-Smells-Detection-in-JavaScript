Boite de dialogue de saisie du Nom d'Utilisateur

Le nom d'utilisateur saisi dans cette boite de dialogue sera ajouté à tous les enregistrements de la base que vous venez d'ouvrir et dont le nom d'utilisateur est vide. Cliquer sur Ok pour ajouter ce nom d'utilisateur ou Cancel pour le laisser tel que.

Vous pouvez également définir ce nom d'utilisateur comme nom par défaut en cliquant sur la case à cocher. Dans ce cas, vous ne verrez pas le nom de vous venez de saisir dans l'écran principal (même si il fera toujours parti des enregistrements), et il sera automatiquement ajouté dans la fenêtre Add pour les nouveaux enregistrements.
