Manage Menu

The Manage Menu is where all of Password Safe's options and settings are controlled.

Change Safe Password

Selecting this option brings up the Change Safe Password dialog.
See Changing Master Passwords for more information.

Make Backup

Safely create a copy of the current password database.
See Backing Up and Restoring Databases for more information.

Restore from Backup

Restore a backed up password database file.
See Backing Up and Restoring Databases for more information.

Options...

Selecting Options... brings up a special tabbed dialog where many of Password Safe's default settings are stored and can be edited. To learn more about a specific tab select it below:

Display Tab
Security Tab
Password Policy Tab
Username Tab
Misc Tab
