Главное окно Password Safe

Главное окно состоит из панели меню, панели инструментов и списка записей в текущей БД.

Панель меню

Панель меню имеет следующие пункты:

File

Для чтения и записи баз данных PasswordSafe. Вы можете создать новую базу, открыть существующую, сохранить текущую (под существующим именем или под новым), экспортировать базу или в старый формат версии 1.7, или в текстовый файл, который может быть открыт текстовым редактором или программой для работы с электронными таблицами.

Open

Выберите этот пункт меню, чтобы открыть другую БД. Вам будет предложено сохранить текущую базу, если это необходимо. Имя текущей базы данных можно увидеть наверху главного окна.

Save

Выберите этот пункт меню, чтобы сохранить текущую базу данных.

Save As

Выберите этот пункт, чтобы сохранить текущую БД под другим именем. Версия под старым именем (без самых последних изменений) все еще будет существовать.

Edit

Для добавления новых записей, редактирования или просмотра существующих, удаления записей и поиска записей, содержащих заданный текст. Также для копирования пароля или имени пользователя выбранной записи в буфер обмена или очистки буфера обмена.

Add Entry

Выберите этот пункт меню, чтобы добавить пароль к текущей базе данных.

Edit/View Entry

Выберите этот пункт меню, чтобы просмотреть или изменить запись.

Delete Entry

Чтобы удалить запись, выделите ее и выберите этот пункт меню.

Copy Password to Clipboard

Чтобы скопировать пароль в буфер обмена, просто кликните дважды по нужной записи. Или можно выделить запись и выбрать пункт меню Copy Password to Clipboard. Затем вы можете вставить пароль в любое приложение Windows. (Если приложение не предоставляет в своем меню возможность вставки из буфера обмена, можно вставить пароль, нажав Control-V или Shift-Insert.)
По умолчанию буфер обмена будет очищен, когда вы выйдете из Password Safe или свернете его.

Copy Username to Clipboard

Чтобы скопировать имя пользователя в буфер обмена, выделите нужную запись и выберите пункт меню Copy Username to Clipboard. Затем вы можете вставить имя пользователя в любое приложение Windows. (Если приложение не предоставляет в своем меню возможность вставки из буфера обмена, можно вставить имя пользователя, нажав Control-V или Shift-Insert.)
По умолчанию буфер обмена будет очищен, когда вы выйдете из Password Safe или свернете его.

Clear Clipboard

Чтобы надежно удалить пароль из буфера обмена, выберите пункт меню Clear Clipboard. Если другая программа скопировала данные в буфер обмена, он не будет очищен.

View

Для выбора того, как будут отображены записи в главном окне: как список или как иерархическое дерево.

Manage

Для изменения сейфовой комбинации (главного пароля) открытой базы данных, создания резервных копий или восстановления БД из резервных копий, а также настройки опций программы.

Change Safe Combination

Выберите этот пункт меню, чтобы изменить сейфовую комбинацию для текущей базы данных. Введите комбинацию в поле "Safe Combination," затем еще раз в поле "verify." Потом нажмите OK. Кнопка OK может быть нажата, если вы ввели одну и ту же комбинацию в обоих полях.

Make Backup

Выберите этот пункт меню, чтобы сделать резервную копию текущей БД.

Restore from Backup

Выберите этот пункт меню, чтобы восстановить сделанную ранее резервную копию базы данных. Восстановленная копия не будет иметь имени. Вам нужно будет дать ей имя в диалоговом окне Save As, если вы хотите сохранить эту базу.

Options

Диалоговое окно Options описывается здесь.

Help

Help - Для доступа к справке и для показа версии программы PasswordSafe.

Панель инструментов

Панель инструментов состоит из кнопок, позволяющих получить быстрый доступ к самым часто используемым функциям. Наведите курсор на кнопку, чтобы получить краткое описание того, для какой функции она предназначена.

Записи

В зависимости от того, какой вид представления главного окна выбран, записи отображаются или как простой список, или как дерево. При отображении списка записи можно сортировать по отображаемым полям, щелкая по заголовку нужной колонки. При отображении в виде дерева записи можно сортировать по категориям, подкатегориям и т. д., в этом случае предоставляется возможность группировать взаимосвязанные пароли вместе.
